# songr

- Description: App that allows user to save a collection of albums - including album title, artist, song count, album length, and cover art to a postgres database of related album and song data.

## Built With
* [Java](https://www.java.com/en/)
* [Gradle](https://gradle.org/)
* [Spring MVC](https://docs.spring.io/spring/docs/current/spring-framework-reference/web.html)
* [PostgreSQL](https://www.postgresql.org/)
